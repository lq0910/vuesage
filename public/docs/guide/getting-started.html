快速开始

安装

VueSage 提供两种安装方式：

全局安装

如果你想在任何项目中使用 VueSage，推荐全局安装：

npm install -g vuesage

项目内安装

如果你只想在特定项目中使用：

npm install --save-dev vuesage

Cursor 编辑器集成

VueSage 可以与 Cursor 编辑器无缝集成，提供实时代码分析和优化建议。

1. 配置 MCP

在项目根目录创建 mcp.json 文件：

{
  "name": "vuesage",
  "version": "1.1.6",
  "description": "Vue 组件智能分析与优化引擎",
  "type": "service",
  "transport": "stdio",
  "command": "vuesage",
  "runtime": "node",
  "global": true,
  "platforms": ["cursor", "vscode"]
}

2. 使用命令

在 Cursor 编辑器中，你可以使用以下命令：

  • /analyze - 分析当前 Vue 组件
  • /fix - 自动修复检测到的问题
  • /optimize - 优化当前组件
  • /report - 生成代码质量报告

命令行使用

分析单个文件

vuesage analyze path/to/component.vue

批量分析

vuesage analyze "src/**/*.vue"

自动修复

vuesage fix path/to/component.vue

生成报告

vuesage analyze path/to/component.vue --report --report-format html

你可以在 这里 查看示例报告。

配置文件

在项目根目录创建 .vuesagerc.json 文件来自定义分析规则：

{
  "rules": {
    "template": {
      "max-attributes-per-line": 3,
      "no-unused-vars": "error",
      "require-v-for-key": "error"
    },
    "script": {
      "max-lines-per-function": 50,
      "no-unused-vars": "error",
      "prefer-const": "warn"
    },
    "style": {
      "require-scoped": "error",
      "no-unused-selectors": "warn"
    }
  },
  "ignore": [
    "node_modules/**",
    "dist/**"
  ]
}
