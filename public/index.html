MCP 配置示例

mcp.json 
{
  "name": "vuesage",
  "version": "1.1.6",
  "description": "Vue 组件智能分析与优化引擎",
  "type": "service",
  "transport": "stdio",
  "command": "vuesage",
  "runtime": "node",
  "global": true,
  "platforms": ["cursor", "vscode"],
  "capabilities": {
    "analyze": {
      "description": "分析 Vue 组件代码质量",
      "input": {
        "type": "object",
        "properties": {
          "component": {
            "type": "string",
            "description": "Vue 组件代码"
          }
        }
      }
    }
  }
}

优化示例 
// 优化前
computed: {
  filteredItems() {
    return this.items
      .filter(item => item.active)
      .map(item => item.value)
      .reduce((acc, val) => acc + val, 0)
  }
}

// 优化后
computed: {
  activeItems() {
    return this.items.filter(item => item.active)
  },
  filteredItems() {
    return this.activeItems
      .map(item => item.value)
      .reduce((acc, val) => acc + val, 0)
  }
}

我已经完成以下优化：

  • 拆分计算属性，避免重复计算
  • 合并了重复的 watch 监听器
  • 统一使用 v-show 优化条件渲染

快速开始

1. 安装

npm install -g vuesage

2. 在 Cursor 中配置

cursor extension install vuesage

3. 开始使用

在 Cursor 中打开 Vue 文件，使用以下命令：

  • • /analyze - 分析当前文件
  • • /fix - 修复检测到的问题
  • • /chat - 开始 AI 对话
